⚠️ A Note on File Permissions
Since your WSL is on a USB, if you created folders while logged in as root, your new user might not be able to "see" or "write" to them. If Lutris gives you permission errors, run this (as root) to give your user ownership of the game folder:

# Run this once while logged in as root
sudo chown -R grimf:grimf /home/grimf/
